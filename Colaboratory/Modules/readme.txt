Modules for Google Colaboratory

In order to use one module in google colaboratory
Example given to jcalc.py

   1) Locate the raw view of the module in GitHub
         https://raw.githubusercontent.com/R6500/Python-bits/master/Colaboratory/Modules/jcalc.py
   2) Copy the URL location
   3) Open a code window in colaboratory
   4) Remove old versions of the module and reload it
         !rm jcalc.*
         !wget https://raw.githubusercontent.com/R6500/Python-bits/master/Colaboratory/Modules/jcalc.py
   5) Now you can use the module
         import jcalc

  jcalc : Miscelaneous calculation and plotting functions
